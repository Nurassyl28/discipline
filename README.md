# Бот для ежедневных опросов

Telegram-бот, который каждый день отправляет опрос в группу. Опросы можно менять через команды прямо в чате.

## Быстрый старт

### 1. Получить токен бота

Напиши [@BotFather](https://t.me/BotFather) → `/newbot` → получи токен.

### 2. Узнать свой Telegram ID

Напиши [@userinfobot](https://t.me/userinfobot) — он пришлёт твой ID.

### 3. Настроить переменные окружения

```bash
cp .env.example .env
```

Заполни `.env`:
```
BOT_TOKEN=1234567890:AAF...
ADMIN_IDS=123456789
TIMEZONE=Asia/Almaty
```

### 4. Установить зависимости и запустить

```bash
pip install -r requirements.txt
python main.py
```

### 5. Первоначальная настройка в Telegram

1. Добавь бота в группу
2. Дай ему права администратора (чтобы мог писать)
3. Напиши в группе `/setgroup` — бот запомнит эту группу
4. Напиши `/settime` и введи время рассылки (например `09:00`)
5. Готово! Каждый день в указанное время бот будет отправлять опрос

---

## Команды

| Команда | Описание |
|---------|----------|
| `/addpoll` | Добавить новый опрос |
| `/listpolls` | Посмотреть все опросы |
| `/delpoll <id>` | Удалить опрос по ID |
| `/setgroup` | Установить текущую группу для рассылки |
| `/settime` | Изменить время ежедневной рассылки |
| `/sendnow` | Отправить опрос прямо сейчас (для теста) |
| `/status` | Текущие настройки бота |

## Деплой на Railway

1. Зайди на [railway.app](https://railway.app) и создай новый проект из GitHub репозитория
2. В разделе **Variables** добавь переменные из `.env`
3. Railway сам определит Python-проект и запустит `worker: python main.py` из `Procfile`

> ⚠️ На Railway файлы сбрасываются при каждом деплое. После деплоя нужно заново выполнить `/setgroup` и `/settime`. Опросы по умолчанию уже встроены в код.

## Деплой на Render

1. Создай новый **Background Worker** сервис
2. Build Command: `pip install -r requirements.txt`
3. Start Command: `python main.py`
4. Добавь переменные окружения

## Опросы по умолчанию (зал/спорт)

Бот поставляется с 3 готовыми опросами:
- Как прошла тренировка сегодня? 💪
- Выполнил план на сегодня? ✅
- Как самочувствие после зала? 🏋️

Опросы отправляются по кругу — каждый день следующий по списку.
